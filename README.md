## ✨ Features

- 📚 Add books
- ✏️ Edit books
- 🗑 Delete books
- 🔄 Check Out / Return books
- 🔍 Live search
- 📂 Filter books
- ↕ Sort books
- 🌙 Dark / Light mode
- 💾 Local Storage
- 📱 Responsive design
